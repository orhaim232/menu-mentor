# MenuMentor Skills Index

This directory contains specialized AI Skills for the MenuMentor project.

### USED
- `caveman`: Ultra-compressed communication mode. Cuts token usage ~75% while keeping full technical accuracy.
- `menu-practice-generator`: Focuses on building the deterministic memory game question generator without using an LLM at runtime.
- `responsive-learning-ui`: Guides the creation and refactoring of mobile-first, RTL-responsive learning interfaces without text truncation.

### INTERNAL DOCUMENTATION TOOL
- `course-submission-documenter`: Internal tool used to write and format academic/course submission documents safely.

### PLANNED FOR CUSTOMER SIMULATION
- `training-scenario-writer`: Guidelines for creating Waiter training scenarios and customer personas.
- `menu-simulation-feedback`: Guidelines for evaluating the Waiter's conversational accuracy and hospitality tone.

### FUTURE / NOT IN MVP
- `menu-upload-parser`: Parsing uploaded PDF/image menus into structured JSON data.
- `semantic-menu-searcher`: Implementing semantic menu search using embeddings and Supabase pgvector.
